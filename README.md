# GEN Z TECH — La tech accessible à tous

Site vitrine de **Gen Z Tech** — une communauté de développeurs unis par le code.

## ✨ Fonctionnalités

- 🌓 **Mode clair/sombre** avec sauvegarde localStorage
- 🌍 **Sélecteur de langue** (FR / EN / SW / 中文)
- 💓 **Logos technos qui pulsent** (heartbeat)
- 📊 **Compteurs animés** (1200+ étudiants, 25+ projets, 8+ pays, 100% satisfaction)
- 🔝 **Bouton retour en haut**
- 📝 **Formulaire de contact** (action Formspree)
- 🌀 **Animations au scroll**
- 📱 **Responsive**

## 📂 Structure

```
sitegenz/
├── index.html
└── README.md
```

## 🔧 Personnalisation

- **Liens réseaux sociaux** : modifie les `href` dans le footer
- **Membres équipe** : modifie les cartes dans `#team`
- **Images** : remplace les URLs Pexels par tes propres images
- **Formspree** : remplace `action` dans le formulaire par ton ID
- **Texte** : modifie directement le HTML (pas de système de traduction — les boutons de langue sont décoratifs)
